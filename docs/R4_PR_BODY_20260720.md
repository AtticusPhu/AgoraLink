# AgoraLink R4

## Summary

- Changes the fixed Rust native screen-sharing default to 1920x1080, 60 FPS, 22 Mbps, NACK repair, and adaptive quality off.
- Adds the explicit `r4_default` GUI preset while preserving saved legacy preset IDs and values.
- Implements the resolution-first adaptive ladder `Q0 -> Q1 -> Q2 -> Q3 -> Q4 -> E1 -> E2`, with recovery along the exact reverse adjacent path.
- Keeps `auto` hardware selection preferences: Intel QSV with software fallback, and D3D11 conversion with CPU fallback.
- Bundles the validated R4 native runtime in the portable package and verifies its fixed SHA-256 before launch.
- Adds R4 GUI configuration tests, packaging checks, portable evidence, and dual-host GUI smoke collection scripts.

- The playout buffer still has optimization headroom.
- Dynamic QSV bitrate changes may fall back to encoder reconstruction.
- The real-network recovery path was not forced during this release pass.
